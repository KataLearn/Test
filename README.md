# Test

Ввод валидных значений

1. Ввод валидного минимального email-адреса:
1.1. Ввести в поле ввода «e@mail.ru»;
1.2. Нажать на кнопку «Проверить»;
1.3. Выводится alert с текстом «Прекрасный email-адрес».

2.Ввод валидного email-адреса: 
2.1. Ввести в поле ввода «qwerty@gmail.com»; 
2.2. Нажать на кнопку «Проверить»; 
2.3. Выводится alert с текстом «Прекрасный email-адрес».

3.Ввод максимального валидного email-адреса: 
3.1. Ввести в поле ввода «very-very.long.user.email.address@gmail.com»; 
3.2. Нажать на кнопку «Проверить»; 
3.3. Выводится alert с текстом «Прекрасный email-адрес».

4.Ввод максимального валидного email-адреса с допустимыми знаками: 
4.1. Ввести в поле ввода «very?very!long.user#email.address*@gmail.com»; 
4.2. Нажать на кнопку «Проверить»; 
4.3. Выводится alert с текстом «Прекрасный email-адрес».

5.Ввод валидного email-адреса состоящий только из допустимых знаков: 
5.1. Ввести в поле ввода «?!#$%@gmail.com»; 
5.2. Нажать на кнопку «Проверить»; 
5.3. Выводится alert с текстом «Прекрасный email-адрес».

6.Валидация ввода значения в инпут: 
6.1. Ввести в поле ввода «user@mail.ru»; 
6.2. Не нажимать на кнопку «Проверить»; 
6.3. Валидация не происходит.
